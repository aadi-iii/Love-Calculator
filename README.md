# ❤️ Love Calculator (Interactive)

This is a fun project inspired by childhood love percentage games.

Instead of generating a random result, this calculator includes small interactive games. Based on user inputs and performance, it calculates a love percentage.

## ✨ Features
- Enter two names
- Play mini-games
- Get a calculated love percentage
- Fun and interactive experience
